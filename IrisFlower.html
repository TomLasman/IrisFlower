iris = datasets.load_iris()

#4 attributes: sepal length, sepal width, petal length and petal width.
data = iris.data
#3 kind of Iris flowers (Setosa = 0, Versicolourand = 1 Virginica = 2) 
target = iris.target
#names of Iris flowers
target_names = iris.target_names

colors = ['navy', 'green', 'darkorange']
kindOfIris = [0, 1, 2]

figure, ((a1, a2)) = plt.subplots(1,2) # figure get the resolution 

for color, i, target_name in zip(colors, kindOfIris, target_names):
     a1.scatter(data[target == i, 0], data[target == i, 1], color=color, label=target_name)
     
a1.set(xlabel='[cm]', ylabel='[cm]',title='IRIS Sepal Dataset')
a1.legend(loc='best', shadow=False, scatterpoints=1)


for color, i, target_name in zip(colors, kindOfIris, target_names):

a2.set(xlabel='[cm]', ylabel='[cm]',title="IRIS petal Dataset")

What I have learned¶

Bunch - It's a data type in python that let u access dicts like it's an object¶

PCA - principal component analysis - It's commonly used for dimensionality reduction.¶

PCA is defined as an orthogonal linear transformation that transforms the data to a new coordinate system that account for the most variance in the data.¶

LAD - linear Discriminant Analysis - identify attributes for the most variance between classes,using known class labels, PCA is a supervised method.¶

Unsupervised - uses machine learning algorithms to analyze and cluster unlabeled data sets.¶

Supervised - machine learning approach that’s defined by its use of labeled datasets.¶

Using labeled inputs and outputs, the model can measure its accuracy and learn over time.¶
